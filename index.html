Table Header <th> Identifies the contents of a cell as being a header for a group o cells
Table Cell <td> This element defines the individual table cells of an element
Colgroup <colgroup> Represents a group of colums in a table

Video

The <video> tag specifies video, such as a movie clip or other video streams.

Currently, there are 3 supported video formats for the <video> element: MP4, WebM, and Ogg.

Also you can use video for example from YouTube using <iframe>.
